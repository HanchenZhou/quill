# Sample Vault

这是一个用来试 Quill 的示例文件夹。Workspace 模式下用它当根目录。

## 包含什么

- `notes/` — 一些示例笔记
- `drafts/` — 草稿
- 一个非 md 文件（应该被灰色禁用显示）

## 试试看

1. 点左侧 `notes/quick-tour.md` —— 能在右侧打开
2. 试着改右边几行，状态栏会变 `unsaved`
3. ⌘S 保存，变回 `saved`
4. 状态栏右下角点几次切主题
5. 关掉这个文件夹（暂时还没有 close 按钮，可以重启）

## 不能干的事（v0 限制）

- 没有 tab；切文件就替换当前
- 不监听文件外部变更
- 拖多个文件只会打开第一个
