Quill UI 重构笔记

关于色彩、字体和那些被默认值偷走的性格。

今天和 Claude 一起把整个 UI 风格重新梳理了一遍。之前那套全 neutral 色 + Inter 的组合，基本就是默认值的产物 —— 能用，但没有性格。

三个方向

  1. 纸感编辑器 — 暖白底 + 衬线大标题
  2. 极简野性 — 锐角 + 等宽 + 高对比
  3. 现代温润 — 中等圆角 + 暖灰 + 一点色彩
